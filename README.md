### Pre-requisitos 📋

```
* PHP 7 o mayor
* MySQL 5.7 o mayor
```

### Instalación 🔧

_Paquete de XAMPP con Apache y Php_

```
Ejecuta los scripts sql de Data Base
```

_Copia todos los archivos en_

```
C:/xampp/htdocs/
```

_Corre XAMPP_
